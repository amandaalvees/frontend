# **Frontend - Gerenciador de Produtos**

Este é o frontend do projeto de gerenciamento de produtos, desenvolvido com React. Ele se comunica com o backend para permitir a criação, edição, exclusão e visualização de produtos.

## **Funcionalidades**

- **Adicionar produto**: Permite adicionar um novo produto.
- **Editar produto**: Permite editar um produto existente.
- **Excluir produto**: Permite excluir um produto da lista.
- **Paginação**: Exibe os produtos em páginas, com navegação entre elas.

## **Tecnologias Utilizadas**

- **React**: Biblioteca JavaScript para a construção de interfaces de usuário.
- **React Hooks**: Para gerenciamento de estado e efeitos.
- **Axios**: Para fazer requisições HTTP ao backend.
- **CSS**: Estilos da aplicação.
